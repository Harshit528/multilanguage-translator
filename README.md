# 🌐 Multilanguage Translator

A web-based translator app built with **Python** and **Streamlit**.

## ✨ Features
- 📝 Type any text and translate instantly
- 🔤 Auto detect source language
- 🎯 Choose from 12 target languages
- 🌍 Translate to ALL languages at once
- 💻 Clean and beautiful web UI

## 🌍 Supported Languages
English, Hindi, Japanese, Chinese, Spanish,
French, Arabic, German, Korean, Portuguese, Russian, Italian

## 🛠️ How to Run
pip install -r requirements.txt
streamlit run translator.py
